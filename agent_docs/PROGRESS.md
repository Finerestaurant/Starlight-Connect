# 프로젝트 진행 상황 요약

**목표:** K-POP 노래와 관련된 인물들의 관계를 시각화하는 인터랙티브 웹 애플리케이션 PoC(Proof of Concept) 개발.

---
## 2026년 2월 18일

- Frontend-Backend 동작 흐름을 분석하고 5개의 Mermaid 다이어그램을 포함한 포괄적인 흐름도 문서를 작성함. 검색, 초기화, 노드 클릭 시나리오별 상세 흐름 설명 완료.

## 2026년 2월 12일

- ID 체계를 `mbid`로 통일하고 UI/API 버그를 수정. '별자리' UI 테마를 적용했으며, '별자리 발자취' 기능은 불안정하여 롤백 후 보류.

## 2026년 2월 11일

- MusicBrainz API 연결 오류를 해결하고, `requests` 라이브러리 기반으로 데이터 수집 파이프라인의 전체 골격을 구현함. 테스트를 통해 다음 단계는 핵심 파싱 로직 구현임을 확인함.

## 2026년 2월 10일

- 백엔드 아키텍처를 서비스 계층으로 리팩토링하고, 데이터 수집 전략을 MusicBrainz API를 사용하는 하이브리드 방식으로 최종 확정함.

## 2026년 2월 9일

- 3D 시각화를 2D(React Flow)로 전환하고, 프론트엔드 아키텍처를 모듈식으로 리팩토링함.
